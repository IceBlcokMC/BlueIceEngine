# JS_Engine

| 中文 | [English](./README-EN.md) |

## 🔨 从源码编译

在编译之前，确保您已经安装了以下工具链：

-   xmake
-   MSVC 2019 或更高版本 (仅限 Windows)
-   LLVM 15 (附带 Clang、libc++) (仅限 Linux)

```bash
git clone --depth=1 https://github.com/engsr6982/JS_Engine.git
cd JS_Engine
xmake -y
```
